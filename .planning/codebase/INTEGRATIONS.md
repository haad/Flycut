# External Integrations

**Analysis Date:** 2026-03-05

## APIs & External Services

**System Clipboard:**
- NSPasteboard (macOS native pasteboard API)
  - Used for: Reading and writing clipboard contents
  - Method: `[NSPasteboard generalPasteboard]` in `AppController.m`
  - Supported types: NSPasteboardTypeString (text)
  - Polling: Timer-based polling of pasteboard contents via `pollPB:` method

**System Event Capture:**
- ApplicationServices.framework - Carbon Events
  - Used for: Global keyboard hotkey interception
  - Implementation: SGHotKeysLib abstracts Carbon event system
  - Purpose: Enables system-wide hotkey capture without app focus

## Local Storage

**Databases:**
- None external. Uses NSUserDefaults only.

**File Storage:**
- Local filesystem only
  - Preferences stored in ~/Library/Preferences/ via NSUserDefaults
  - No cloud synchronization files stored locally

**Caching:**
- In-memory clipboard history cache
  - Stored in FlycutStore (NSMutableArray in `FlycutStore.h`)
  - Size limit: Configurable via preferences (rememberNum, displayNum)
  - No persistent disk cache by default

## Authentication & Identity

**Auth Provider:**
- None required
- Application is standalone (menu bar app)
- No user accounts or cloud services

**System Permissions:**
- Accessibility permission (required)
  - Used for: Injecting paste events via AppleEvents
  - Requested via NSAppleEventsUsageDescription in Info.plist
  - Enforced by macOS security (Accessibility Preferences)

- Login Item permission (required for startup automation)
  - Managed via ServiceManagement.framework
  - Implements secure login item registration (no RunOnce keys)
  - FlycutHelper executable added as login item

## System Services Integration

**Pasteboard Monitoring:**
- Polling Timer: `pollPBTimer` (user-configurable interval, default ~0.5 seconds)
- Trigger: Timer fires `pollPB:` method
- Check: Compares NSPasteboard changeCount to track new clipboard changes

**Password Detection:**
- Local filtering (no external service)
- Recognizes common password/secure content types:
  - PasswordPboardType
  - org.nspasteboard.TransientType
  - org.nspasteboard.ConcealedType
  - org.nspasteboard.AutoGeneratedType
  - com.agilebits.onepassword

**Duplicate Detection:**
- Local in-memory comparison
- User-configurable via preferences
